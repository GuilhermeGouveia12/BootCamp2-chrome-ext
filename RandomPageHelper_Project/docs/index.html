RPH

Random Page Helper

Extensão para Chrome que abre uma página aleatória de sites educativos ou de curiosidades.

Instalação Manual

  1. Acesse chrome://extensions e ative "Developer mode".
  2. Clique em "Load unpacked" e selecione a pasta do projeto (branch main).

Permissões

  • tabs
  • storage

Download

Release com ZIP disponível no GitHub Releases. Também pode baixar o ZIP diretamente do repositório.

Testes E2E (Playwright)

O repositório inclui configuração para executar testes E2E com Playwright e rodar em CI via GitHub Actions.
